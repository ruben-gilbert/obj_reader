Object File Reader by Ruben Gilbert - CS461 Final Project

This application reads in .obj files and parses them for vertex, texture, and normal data and displays the corresponding object. It uses a simple phong lighting scheme (lighting is normally specified in the material files, but there were complications with this--noted in the summary paper). The mesh is colored based on the (absolute value) of the normal at each vertex. Use the dropdown to select between the different pre-loaded .obj files. The app supports .obj files with triangles or quads. If normals aren't provided in the .obj file, this app will calculate them for you.
